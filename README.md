# HP Laptop Sales Performance Dashboard (Excel)
## 📊 Project Overview
This project analyzes HP laptop sales performance to track revenue, profit, units sold, and regional performance using Excel dashboards.

🚀 Features
---

KPI Cards (Revenue, Profit, Units Sold, Margin)  
Monthly Revenue Trend  
Revenue vs Profit Comparison  
Model-wise Revenue  
Region-wise Sales  
Store Type Share  
Dark/Light Mode Toggle  
Interactive Slicers  

Tools Used
--  
Microsoft Excel  
Pivot Tables  
Power Query  
Charts  
Slicers
